<h1 align="center"><b>I Want Links on YouTube Video</b></h1>

**I Want Links on YouTube Video** is a free and open-source browser extension that helps you quickly find and view all links shared within YouTube video descriptions or comments — directly from the extension itself.

---

### What is it?

This extension makes it easy to access external links related to any YouTube video without manually opening the description.  
It’s especially useful for videos that include multiple links to sources, products, or references.

---

### How does it work?

Once installed, simply open a YouTube video.  
The extension automatically detects and displays all available links associated with that video. You can:

- View all detected links in one clean list  
- Click any link to open it in a new tab  
- Access the feature directly from the extension popup

### Message Meanings

The extension may show different messages based on the current page or video status:

| Message | Meaning |
|----------|----------|
| **You're not visiting YouTube!** | The extension only works on YouTube. Open a YouTube video page to use it. |
| **No found video.** | No video was detected on the current YouTube page, or if a video exists, the link didn’t exist in the JSON URL source. |
| **Found video!** | A YouTube video has been successfully detected and is ready for link extraction. |
